## About Me

I'm a **Systems Analyst at INSEAD** with 3 years of experience turning real institutional needs into working software — full-stack apps (Next.js, TypeScript, Prisma, Supabase), production tooling, and a hand in modernizing legacy platforms end to end.

I've been **building automations since 2020**, and AI didn't replace that skill — it multiplied it. A single n8n pipeline I built a year ago now runs multiple content channels on autopilot. I treat AI the same way I treat any powerful tool: something to be directed with intent, not leaned on blindly — and I actively coach others (from near-zero AI experience to shipping functional MVPs) on exactly that mindset. Self-taught, and it works.

On the freelance side, I took a client's ticket management system apart and rebuilt it from scratch — same core need, dramatically better UX.

I'm also currently **expanding into Salesforce development** (Apex, jsforce) — early days, actively leveling up, no pretending otherwise.

<br/>

## What I'm Building

- **Production tooling @ INSEAD** — full-stack apps and platform modernization work that real teams depend on
- **Client project: ticket management rebuild** — full UX and architecture overhaul, from legacy pain points to a modern experience
- **Multi-channel content automation** — one n8n pipeline, running since 2024, powering multiple channels hands-off
- **AI-native workflows** — Claude Code as an execution layer, multi-agent pipelines, cost-optimized LLM stacks (DeepSeek, MiniMax), MCP servers
- **Islamic finance & halal-compliant investing** — exploring automation and tooling in this space
